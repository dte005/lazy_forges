Terminal database schema IDE

LazyForge

Forge database schemas without leaving your terminal.

Como começar Ver mídia/brand kit

O que é

O LazyForge é uma aplicação TUI (Terminal User Interface) em Dart para modelagem de schema de banco de dados em fluxo keyboard-driven. Inspirado por ferramentas como LazyGit/LazyVim, permite criar tabelas, colunas e relacionamentos por comandos, com feedback visual imediato no terminal.

Como rodar

Durante desenvolvimento local:

dart pub get
dart run bin/lazy_forge.dart

Com hot reload no terminal:

dart --enable-vm-service bin/lazy_forge.dart

Instalação global (Dart SDK)

Objetivo: rodar com o comando lazy_forge em qualquer diretório. O diretório atual do terminal vira a base de trabalho para salvar projetos e exports.

Via Git

dart pub global activate --source git <URL_DO_REPOSITORIO>
lazy_forge

Via pub.dev

dart pub global activate lazy_forge
lazy_forge

Em Windows, confirme que %LOCALAPPDATA%\Pub\Cache\bin está no PATH.

Comandos principais

create table users --autoincrement
add columns users id text as pk; name text; status enum options(active|inactive)
add fk users role_id references roles id
rename column users name to full_name
delete table users
export

Fluxo de uso

  • Criação/abertura de projeto com seleção de banco (postgres/mysql/sqlite).
  • Visualização de schema no editor (tabelas + relações PK/FK).
  • Persistência de projetos no diretório atual de execução.
  • Export DDL SQL no diretório atual de execução.
